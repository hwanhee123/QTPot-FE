  서비스 소개
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

QTPot은 매일 큐티(QT, Quiet Time) 사진을 업로드하여
성도님들과 말씀 묵상을 함께 나누는 인증 서비스입니다.

  • 매일 큐티 사진을 올려 하루를 기록합니다.
  • 피드에서 다른 성도님들의 묵상을 보고 댓글로 격려합니다.
  • 매월 목표 일수를 채우면 단계별 뱃지를 획득합니다.
  • 월간 랭킹으로 서로 동기부여합니다.
  • 모바일 앱처럼 홈 화면에 추가해 사용할 수 있습니다(PWA).


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  1. 회원가입 / 로그인
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

▶ 회원가입
  - 이름, 이메일, 비밀번호(8자 이상)를 입력합니다.
  - 가입 후 바로 로그인이 가능합니다.

▶ 로그인
  - 가입한 이메일과 비밀번호로 로그인합니다.
  - 로그인 상태는 24시간 유지됩니다.

▶ 비밀번호 분실
  - 로그인 화면 하단의 "비밀번호 찾기"를 누릅니다.
  - 가입 시 등록한 이름과 이메일을 입력하면
    임시 비밀번호(123456789)로 초기화됩니다.
  - 초기화 후 반드시 "내 기록 > 비밀번호 변경"에서
    새 비밀번호로 변경하세요.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  2. 메뉴 안내
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  🏠 피드       : 모든 성도님들의 큐티 인증 피드 및 댓글
  📅 대시보드   : 내 큐티 인증 업로드 및 월별 달력 확인
  🏆 랭킹       : 월별 인증 횟수 순위 확인
  👤 내 기록    : 내 통계, 뱃지, 알림 설정, 비밀번호 변경
  ⚙️ 관리자     : (관리자 전용) 멤버 관리 및 날짜별 인증 현황

  ※ 모바일에서는 화면 하단 탭바로 이동합니다.
  ※ PC에서는 상단 메뉴바로 이동합니다.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  3. 큐티 인증하기  (대시보드)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

① "대시보드"로 이동합니다.
② 인증 카드에서 사진을 선택합니다.
   - 여러 장 동시 업로드 가능 (JPG, PNG / 장당 최대 5MB)
   - 선택한 사진을 클릭하면 목록에서 제거됩니다.
   - (+) 버튼으로 사진을 추가 선택할 수 있습니다.
③ 비공개 여부를 설정합니다. (토글 스위치)
   - 비공개 ON  : 피드에서 다른 사람에게 이미지가 흐릿하게 표시됩니다.
                  본인 대시보드에서는 원본으로 확인 가능합니다.
   - 비공개 OFF : 피드에 이미지가 그대로 공개됩니다. (기본값)
④ 한줄 소감을 선택적으로 입력합니다.
⑤ "인증샷 제출" 버튼을 누릅니다.

▶ 하루 1회만 인증 가능합니다.
▶ 과거 날짜 인증 : 달력에서 해당 날짜를 클릭한 뒤 업로드합니다.
▶ 업로드 후 소감은 수정할 수 있으며, 게시물 삭제도 가능합니다.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  4. 피드
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

- 기본적으로 오늘 날짜의 인증 목록을 보여줍니다.
- 날짜를 변경하거나 "전체보기"로 모든 기록을 확인할 수 있습니다.

▶ 비공개 게시물
  - 게시물 작성자가 비공개로 설정한 경우,
    다른 사람에게는 이미지가 흐릿하게 표시됩니다.
  - 본인 게시물은 항상 원본으로 표시됩니다.
  - 이름 옆에 "비공개" 표시가 함께 나타납니다.

▶ 댓글 기능
  - 각 게시물 하단의 "💬 댓글 보기"를 눌러 댓글 창을 엽니다.
  - 입력창에 댓글을 작성하고 "등록" 버튼을 누릅니다. (최대 200자)
  - 본인이 작성한 댓글은 삭제할 수 있습니다.
  - 내 게시물에 댓글이 달리면 푸시 알림을 받을 수 있습니다.
    (알림 수신을 위해 "내 기록 > 알림 설정"에서 허용 필요)

▶ 소감 수정 / 게시물 삭제
  - 본인 게시물에만 "소감 수정"과 "삭제" 버튼이 표시됩니다.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  5. 대시보드 — 달력 & 기록 보기
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

- 인증한 날짜는 달력에 색깔 원으로 표시됩니다.
- 날짜를 클릭하면 해당 날의 인증 사진을 하단에서 확인합니다.
- 좌우 화살표(< >)로 이전/다음 달을 조회합니다.
- 진행률 바로 이번 달 목표 달성도를 한눈에 확인합니다.
- "전체보기" 버튼으로 내 모든 인증 기록을 날짜 순으로 확인합니다.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  6. 랭킹
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

- 이번 달 인증 횟수를 기준으로 순위를 표시합니다.
- 상단의 연도/월 선택기로 과거 월 랭킹도 확인할 수 있습니다.
- 🥇 1위, 🥈 2위, 🥉 3위는 별도로 강조 표시됩니다.

  7. 뱃지 시스템
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

매월 1일 자정, 전달 인증 횟수를 기준으로 뱃지가 자동 지급됩니다.

  🌰 씨앗  : 전달 10일 이상 인증
  🌱 새싹  : 전달 20일 이상 인증
  🌸 꽃    : 전달 30일 이상 인증

※ 가장 높은 단계 뱃지 1개만 지급됩니다.
   예) 30일 달성 → 꽃 뱃지만 지급 (씨앗·새싹은 별도 지급 안 됨)
※ 같은 달에 뱃지가 중복 지급되지 않습니다.
※ 지급이 늦을 경우 "내 기록" 페이지를 방문하면 자동으로 확인됩니다.
※ 획득한 뱃지는 "내 기록" 페이지에서 전부 확인할 수 있습니다.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  8. 내 기록 (프로필)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

▶ 인증 통계
  - 이번 달 / 올해 / 누적 인증 횟수를 카드로 확인합니다.
  - 이번 달 인증 횟수에 따라 달성 단계가 표시됩니다.
    (🌰 씨앗 10일 / 🌱 새싹 20일 / 🌸 꽃 30일)

▶ 획득한 뱃지
  - 지금까지 받은 모든 뱃지 목록과 취득 월을 확인합니다.

▶ 알림 설정
  - "알림 허용" 버튼을 누르면 브라우저 알림 권한을 요청합니다.
  - 허용 후에는 다른 사람이 내 게시물에 댓글을 달면
    푸시 알림(앱 알림)이 전송됩니다.
  - 알림 아이콘 : 🔔 켜짐 / 🔕 꺼짐
  - 스마트폰에서 홈 화면에 추가(PWA)한 경우 더욱 잘 작동합니다.

▶ 비밀번호 변경
  - 새 비밀번호(8자 이상)를 두 번 동일하게 입력 후 변경합니다.

  9. 관리자 기능  (관리자 계정 전용)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

관리자 계정으로 로그인하면 메뉴에 "⚙️ 관리자" 탭이 나타납니다.

▶ 멤버 관리 탭
  - 전체 멤버 목록을 확인합니다.
  - 각 멤버별 이번 달 / 올해 / 누적 인증 횟수를 표로 확인합니다.
  - 역할(USER / LEADER / ADMIN)을 확인합니다.
  - 비밀번호 초기화 :
    해당 멤버 행의 "초기화" 버튼을 누르면
    비밀번호가 임시값(123456789)으로 초기화됩니다.
    초기화 후 해당 멤버에게 비밀번호 변경을 안내해주세요.

▶ 날짜별 인증 탭
  - 날짜를 선택하고 "조회" 버튼을 누르면
    해당 날짜에 인증한 전체 멤버의 사진 목록을 확인합니다.
  - 상단에 "총 N명 인증"으로 인원 수가 표시됩니다.

  자주 묻는 질문 (FAQ)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q. 하루에 두 번 인증할 수 있나요?
A. 하루 1회만 가능합니다. 재업로드가 필요하면
   기존 게시물을 삭제한 후 다시 업로드해주세요.

Q. 과거 날짜에 인증을 올릴 수 있나요?
A. 가능합니다. 대시보드 달력에서 원하는 날짜를 클릭하면
   해당 날짜로 인증 업로드가 됩니다.

Q. 비공개로 올리면 어떻게 되나요?
A. 피드에서 다른 사람에게는 이미지가 흐릿하게 표시됩니다.
   본인 대시보드에서는 원본 이미지를 그대로 볼 수 있습니다.
   비공개 여부는 업로드 시점에만 설정할 수 있습니다.

Q. 댓글 알림이 오지 않아요.
A. "내 기록 > 알림 설정"에서 "알림 허용" 버튼을 먼저 눌러주세요.
   브라우저에서 알림을 허용해야 푸시 알림이 전송됩니다.
   스마트폰에서는 홈 화면에 앱을 추가(PWA)한 경우 더 안정적입니다.

Q. 뱃지는 언제 지급되나요?
A. 매월 1일 자정에 전달 실적 기준으로 자동 지급됩니다.
   지급이 안 된 경우 "내 기록" 페이지를 열면 자동으로 확인됩니다.
   10일·20일·30일 중 가장 높은 단계 뱃지 1개만 지급됩니다.

Q. 사진 업로드가 안 돼요.
A. - 파일 크기를 확인해주세요. (장당 최대 5MB)
   - JPG 또는 PNG 형식만 지원합니다.
   - 해당 날짜에 이미 인증한 경우 재업로드가 불가합니다.

Q. 비밀번호를 잊어버렸어요.
A. 로그인 화면의 "비밀번호 찾기"를 이용하거나
   관리자에게 문의하면 임시 비밀번호(123456789)로 초기화됩니다.
   초기화 후 반드시 "내 기록 > 비밀번호 변경"에서 변경하세요.

Q. 모바일에서 앱처럼 쓸 수 있나요?
A. 가능합니다. 모바일 브라우저에서 사이트 접속 후
   "홈 화면에 추가"를 선택하면 앱 아이콘으로 설치됩니다.
   설치 후에는 푸시 알림도 더 안정적으로 수신됩니다.

  관리자 참고 사항
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

▶ 회원 역할(Role) 종류
  - USER   : 일반 회원 (기본값)
  - LEADER : 리더 — 월별 전체 인증 현황 조회 가능
  - ADMIN  : 관리자 — 멤버 관리, 비밀번호 초기화, 날짜별 조회 가능
  ※ 역할 변경은 데이터베이스에서 직접 수정합니다.

▶ 비밀번호 초기화 임시값 : 123456789
  초기화 후 해당 멤버에게 반드시 비밀번호 변경을 안내해주세요.

▶ 뱃지 자동 지급 스케줄
  매월 1일 00:00:00 서버 시간 기준으로 실행됩니다.
  서버가 해당 시각에 다운된 경우, 멤버가 "내 기록" 페이지를
  방문하면 자동으로 보완 지급됩니다.

▶ 이미지 저장
  업로드된 이미지는 AWS S3에 저장됩니다.
  게시물 삭제 시 S3 이미지도 함께 삭제됩니다.
